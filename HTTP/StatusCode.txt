100 -> Continue

200 -> OK (GET, POST)
201 -> Created (PUT)
204 -> No Content (DELETE, PUT)

300 -> Moved Permanently
308 -> Permanent Redirect
302 -> Found
307 -> Temporary Redirect

400 -> Bad Request 
401 -> Unauthorized 
404 -> Not Found
405 -> Method Not Allowed

500 -> Internal Server Error
503 -> Service Unavaiable
